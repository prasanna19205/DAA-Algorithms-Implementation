# 📘 Design and Analysis of Algorithms (DAA) Project

## 📌 Overview
This project contains implementations of fundamental algorithms in C, covering three major design techniques:
- Greedy Method
- Divide and Conquer
- Dynamic Programming

The goal is to understand algorithm design, analyze complexity, and test performance with different inputs.

---

## 🧠 Algorithms Implemented

### 🔹 1. Greedy Algorithm – Activity Selection
- Select maximum number of non-overlapping activities
- Strategy: Choose activity with earliest finish time
- Time Complexity: O(n log n)

---

### 🔹 2. Divide and Conquer – Closest Pair of Points
- Find the closest pair of points in 2D plane
- Implemented using brute-force approach
- Time Complexity: O(n²)
- (Optimized version possible: O(n log n))

---

### 🔹 3. Dynamic Programming – Edit Distance
- Convert one string into another with minimum operations
- Operations: Insert, Delete, Replace
- Time Complexity: O(n × m)

---
