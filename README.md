# Design and Analysis of Algorithms Project

## 📌 Description
This project contains implementations of fundamental algorithms in C.

## 🔹 Algorithms Covered
1. Greedy Algorithm – Activity Selection
2. Divide and Conquer – Closest Pair of Points
3. Dynamic Programming – Edit Distance

## ⚙️ How to Run

Compile:
gcc filename.c -o output

Run:
./output

## 📊 Features
- Well-commented code
- Time complexity analysis
- Multiple test cases
